# Empleados

## TECNOLOGÍAS
    - Angular
    - CSS (Flexbox)
    - BOOSTRAP 5

## FUNCIONALIDAD

    - App que permite al usuario añadir empleados (nombre, apellidos, dni y especialidad)

    - Muestra los datos en una lista y permite añadir un campo de característica dentro de los datos de la lista
